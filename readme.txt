== Description ==

This is the Beautiful responsive WP Page Builder Portfolio. Display Portfolio OR Projects in a grid view.

Showcase your portfolio. Rhino Portfolio is the Custom Portfolio post type. Add portfolio items with graphics and other details to showcase them in a conversion-friendly way.

== Installation ==

The automatic installation is the easiest way to install any plugin in WordPress. You can perform an automatic installation of Rhino Portfolio by logging in to your WordPress dashboard, navigating to the “Plugins” menu and click on the “Add New” button.
This will open up a page showing all the available plugins in WordPress. In the search field, type Rhino Portfolio by Fahim or search just "Fahim". The search result will show you Rhino Portfolio plugin, you can then see the detailed info by clicking on “More Details” and to install just click on the “Install Now” button.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Plugin Name screen to configure the plugin.
4. Must install WP Page Builder with this plugin.
5. After installing the plugin, go to the "Settings" > Permalinks > click "Save Changes".
